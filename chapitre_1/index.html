Liens hypertextes

Retour à la page d'accueil avec les balises <a>texte</a>

<a href="index.html" accesskey="a" title="accueil" target="_blank">Retour à la page d'accueil</a>

  • href="index.html" : indique l'adresse de la page à afficher
  • accesskey="a" : ici permet d'activer le lien en appuyant sur "alt" + la touche "a"
  • title="accueil" : affiche un texte info-bulle au passage de la souris
  • target="_blank" : ici signifie ouvrir le lien dans une nouvelle fenêtre
      • Autre valeurs possibles :
    • target="_blank" : ouvre dans une nouvelle fenêtre (a privilégier)
    • target="_parent" : affiche la page dans la fenêtre parente
    • target="_self" : affiche le lien dans la même fenêtre
    • target="_top" : revient à la fenêtre parente de plus haut niveau

Listes de définitions

Permet de présenter une suite de termes accompagnés de leur définition

<dl>
description list = liste de descriptions (encadre la liste contenant les termes et leur définition)
<dt>
description list term = un terme de la liste de desciptions
<dd>
description list description = une description de la liste de descriptions (généralement associée à un terme)

Utilisé aussi pour afficher des dialogues, la balise <dt> mentionnant le nom du personnage et la balise <dd> encadrant sa réplique.

Exemple :

Chez le médecin

Le patient
Docteur, je suis amnésique...
Le docteur
Ah ! Depuis quand ?
Le patient
Depuis quand, quoi ?
